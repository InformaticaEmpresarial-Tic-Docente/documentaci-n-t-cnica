Introducción a HTML5

HTML5 es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML.

Esta versión introduce numerosas características nuevas que facilitan la creación de aplicaciones web más dinámicas e interactivas.

  • Nuevos elementos semánticos como <header>, <footer>, <section>, etc.
  • Nuevos atributos para formularios como placeholder, required, pattern
  • Soporte nativo para audio y video sin necesidad de plugins
  • Canvas para gráficos y animaciones
  • Almacenamiento local mejorado
<!DOCTYPE html> <html lang="es">

HTML5 está diseñado para ser compatible con versiones anteriores y funciona en todos los navegadores modernos.

Estructura básica

Todos los documentos HTML5 deben comenzar con una declaración <!DOCTYPE html>.

La estructura básica consta de:

  • Elemento raíz <html>
  • Sección <head> con metadatos
  • Sección <body> con el contenido visible
<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <title>Título del documento</title> </head> <body> <!-- Contenido --> </body> </html>

El atributo lang especifica el idioma del documento para accesibilidad y SEO.

Etiquetas semánticas

HTML5 introduce elementos semánticos que describen mejor su significado tanto para el navegador como para el desarrollador.

Estas etiquetas mejoran la accesibilidad y el SEO:

  • <header> - Cabecera de sección o página
  • <nav> - Contiene enlaces de navegación
  • <section> - Sección temática del contenido
  • <article> - Contenido independiente y autónomo
  • <footer> - Pie de página o sección
<article> <header> <h2>Título del artículo</h2> </header> <p>Contenido del artículo...</p> </article>

El uso correcto de estas etiquetas mejora la estructura del documento y su indexación.

Multimedia

HTML5 proporciona elementos nativos para multimedia sin necesidad de plugins externos.

Los principales elementos son:

  • <video> - Para incrustar contenido de video
  • <audio> - Para incrustar contenido de audio
  • <canvas> - Para dibujar gráficos con JavaScript
  • <svg> - Para gráficos vectoriales escalables
<video width="320" height="240" controls> <source src="movie.mp4" type="video/mp4"> Tu navegador no soporta el elemento video. </video> <audio controls> <source src="audio.ogg" type="audio/ogg"> Tu navegador no soporta el elemento audio. </audio>

Estos elementos admiten múltiples formatos y son altamente personalizables.

Formularios avanzados

HTML5 introduce nuevos tipos de entrada y atributos para formularios.

Nuevos tipos de input:

  • email, url, tel - Para datos específicos
  • number, range - Para valores numéricos
  • date, time, datetime-local - Para fechas y horas
  • color - Selector de color
  • search - Para campos de búsqueda
<input type="email" name="user_email" required> <input type="date" name="fecha_nacimiento">

Nuevos atributos como placeholder, required, pattern y autocomplete mejoran la experiencia del usuario.

API HTML5

HTML5 incluye varias APIs poderosas para aplicaciones web modernas.

APIs principales:

  • Geolocalización - Acceso a la ubicación del usuario
  • Drag and Drop - Arrastrar y soltar elementos
  • Web Storage - Almacenamiento local y de sesión
  • Web Workers - Ejecución de scripts en segundo plano
  • Web Sockets - Comunicación bidireccional con el servidor
// Ejemplo de geolocalización navigator.geolocation.getCurrentPosition( function(position) { console.log(position.coords.latitude, position.coords.longitude); } );

Estas APIs permiten crear aplicaciones web ricas y complejas.

Buenas prácticas

Para desarrollar con HTML5 de manera profesional, sigue estas buenas prácticas:

  • Usa la estructura semántica correcta
  • Valida tu código regularmente
  • Optimiza el rendimiento de carga
  • Prioriza la accesibilidad
  • Documenta tu código
<!-- Buen ejemplo de comentario --> <section id="comentarios"> <h2>Comentarios</h2> <article class="comentario"> <p>Contenido del comentario...</p> </article> </section>

El código limpio y bien estructurado es más fácil de mantener y escalar.
